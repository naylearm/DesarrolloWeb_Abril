  8. <form>

    9. Es un contenedor para diferentes tipos de elementos de entrada como: campos de texto, casillas de verificación, botones de envñio, etc.

    • <input type="text">


      • Muestra un campo de entrada de texto de una sola linea.


    • <input type="radio">


      • Muestra un botón de radio, para seleccionar una de las muchas opciones.


    • <input type="checkbox">


      • Muestra una casilla de verificación, para seleccionar cero o más opciones.


    • <input type="submit">


      • Muestra un botón de envío, para enviar el formulario.

  13. <img>

    14. Se utiliza para incrustar una imagen en una pagona HTML.
    Tecnicamente, las imagenes no se insertan en una pagina web, las imagenes estan vinculadas a paginas web. la etiqueta <ima> crea un espacio de espera para la imagen de referencia.
    Tiene atributos obligatorios:

    • src: especifica la ruta a la imagen
    • alt: especifica un texto alternativo para la imagen, si la imagen por algun motivo no se puede mostrar


    Ejemplo img

  14. <map>

    15. Se utiliza para definir un mapa de imagen, es una imagen con areas en las que se puede hacer clic.
    El atributo de nombre requerido del <map> esta asociado con el atributo usemap de <img> y crea una relacion entre la imagen y el mapa.

    Ejemplo map
